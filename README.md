# USD–INR Exchange Rate Analysis (2016–2026)

This project analyzes the relationship between the USD–INR exchange rate and key macroeconomic indicators from 2016 to 2026, including Gold Prices, Crude Oil Prices, and Repo Rates. Data was collected from publicly available sources, explored and analyzed using Jupyter Notebook, cleaned and transformed using SQL, exported into CSV format, and visualized through interactive Power BI dashboards. The project provides insights into exchange rate trends, market movements, and the impact of economic factors on currency fluctuations over time.

### Tools Used: Python (Pandas), Jupyter Notebook, SQL, Power BI, CSV

### Key Analysis:

* USD–INR exchange rate trends (2016–2026)
* Correlation between exchange rates and gold prices
* Impact of crude oil price fluctuations
* Repo rate trend analysis
* Interactive Power BI dashboards and KPI visualizations
